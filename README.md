# Siscad-crawler
Node.js Siscad(https://siscad.ufms.br/) crawler.

## Usage

Set your siscad login and password in function logarSiscad or export.

Then:

    $ node siscad.js
or:

    $ nodemon siscad.js

## Como usar

Preencha a função logarSiscad(login, senha) ou exporte.

Em seguida:

    $ node siscad.js
ou:

    $ nodemon siscad.js
